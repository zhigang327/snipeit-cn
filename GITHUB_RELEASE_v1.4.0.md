# Snipe-CN v1.4.0 发布说明

## 发布日期
2025-03-12

## 版本亮点
🎉 **资产借用管理功能正式发布！**

本次更新带来了完整的资产借用管理流程，支持借用申请、审批、借出、归还等全生命周期管理。

---

## 新增功能

### 1. 借用申请和审批流程
- 提交借用申请，填写借用目的、日期、押金等信息
- 管理员审批通过或拒绝申请
- 审批记录完整保留

### 2. 借用状态流转
```
待审批 → 已批准 → 已借出 → 已归还
    ↓        ↓
  已拒绝   逾期未还
    ↓
  已取消
```

### 3. 押金管理
- 支持设置押金金额
- 记录押金退还状态
- 押金收支统计

### 4. 损坏赔偿
- 归还时记录损坏情况
- 填写损坏赔偿金额
- 标记损坏处理状态

### 5. 逾期管理
- 自动检测逾期记录
- 逾期状态自动标记
- 即将到期提醒（默认3天）

### 6. 统计面板
- 待审批数量
- 已借出数量
- 逾期未还数量
- 逾期率统计
- 平均借用时长

### 7. 微信通知集成
- 借用申请提交通知
- 审批结果通知
- 资产借出/归还通知

## 如何升级

### 1. 更新代码
```bash
git pull origin main
```

### 2. 执行数据库迁移
```bash
php artisan migrate
```

### 3. 配置微信通知（可选）
在 `.env` 文件中添加：
```
WECHAT_NOTIFY_BORROW=true
```
