## 三、集成测试建议

### 1. 环境准备
```bash
# 1. 启动Docker容器
docker-compose up -d

# 2. 访问系统
http://localhost:5173

# 3. 登录系统
邮箱: admin@example.com
密码: admin123
```

### 2. 配置步骤
1. 进入 `系统设置` → `微信通知`
2. 获取企业微信群机器人Webhook地址
3. 粘贴Webhook地址到输入框
4. 开启微信通知开关
5. 点击"保存配置"
6. 点击"发送测试消息"

### 3. 验证步骤
1. **基础功能测试**:
   - 发送测试消息，确认微信群接收
   - 测试开关状态切换
   - 验证配置保存功能

2. **集成功能测试**:
   - 创建资产，测试到期提醒
   - 领用/归还资产，测试变动通知
   - 创建盘点任务，测试任务通知
   - 完成盘点任务，测试完成通知

3. **配置功能测试**:
   - 测试各项通知开关功能
