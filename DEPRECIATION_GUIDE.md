# 资产折旧功能指南

## 功能简介

Snipe-CN支持两种常见的资产折旧计算方法:

- **直线法(Straight Line)**: 按使用年限平均折旧
- **双倍余额递减法(Declining Balance)**: 加速折旧方法

## 折旧方法说明

### 1. 直线法

**适用场景**: 适用于价值稳定、使用年限均匀的资产(如办公家具、建筑物等)

**计算公式**:
```
年折旧额 = (原值 - 残值) / 使用年限
月折旧额 = 年折旧额 / 12
折旧率 = 1 / 使用年限
```

**特点**:
- 每年折旧额相同
- 计算简单,易于理解
- 折旧额均匀分布

**示例**:
```
购买价格: ¥10,000
残值: ¥500
使用年限: 5年

年折旧额 = (10,000 - 500) / 5 = ¥1,900
月折旧额 = 1,900 / 12 = ¥158.33
```

### 2. 双倍余额递减法

**适用场景**: 适用于价值递减快、技术更新快的资产(如电脑、手机、服务器等)

**计算公式**:
```
年折旧率 = 2 / 使用年限
本期折旧额 = 账面价值 × 年折旧率
新账面价值 = 账面价值 - 本期折旧额
```

**特点**:
- 前期折旧额大,后期折旧额小
- 考虑货币时间价值
- 更符合资产价值实际变化

**示例**:
```
购买价格: ¥10,000
使用年限: 5年

年折旧率 = 2 / 5 = 40%

第一年: 10,000 × 40% = ¥4,000
第二年: 6,000 × 40% = ¥2,400
第三年: 3,600 × 40% = ¥1,440
...
```

## 使用指南

### 配置资产折旧

1. 进入"资产折旧"页面
2. 找到需要配置折旧的资产
3. 点击"配置折旧"按钮
4. 填写以下信息:
   - **折旧方法**: 选择直线法或双倍余额递减法
   - **残值**: 资产报废时的预计价值
   - **使用年限**: 资产的使用年限(年)
5. 点击"保存"

### 执行折旧计算

#### 单个资产折旧

1. 在资产列表中找到要折旧的资产
2. 点击"执行折旧"按钮
3. 确认操作
4. 系统自动计算折旧并记录

#### 批量折旧

1. 点击"批量折旧"按钮
2. 选择折旧日期
3. 选择要折旧的资产(可多选)
4. 点击"执行批量折旧"

### 查看折旧详情

1. 点击资产的"折旧详情"按钮
2. 查看以下信息:
   - 折旧方法
   - 累计折旧
   - 当前账面价值
   - 折旧进度
   - 折旧预测表

### 折旧预测表

系统会自动生成折旧预测表,显示资产在未来几年的折旧情况:

## 折旧状态

资产折旧有以下几种状态:

1. **未配置**: 资产未设置折旧参数
2. **折旧进行中**: 资产正在折旧,未到达残值
3. **已全额折旧**: 累计折旧已达到(原值 - 残值)

## 注意事项

### 1. 折旧时机

- 建议每月执行一次折旧
- 可使用批量折旧功能提高效率
- 折旧日期通常为每月最后一天

### 2. 数据一致性

- 折旧操作不可撤销
- 执行前请确认数据正确
- 建议定期备份数据库

### 3. 财务合规

- 折旧方法应符合企业会计制度
- 使用年限应与税法规定一致
- 残值比例一般不低于5%

### 4. 资产状态

- 资产报废后应停止折旧
- 资产出售时应确认折旧总额
- 盘点发现的损坏资产需调整折旧

## API说明

### 计算折旧
```
GET /api/assets/{asset_id}/depreciation/calculate
```

### 执行折旧
```
POST /api/assets/{asset_id}/depreciation/execute
Content-Type: application/json

{
  "depreciation_date": "2025-03-31",
  "notes": "2025年3月折旧"
}
```

### 批量折旧
```
POST /api/depreciation/batch
Content-Type: application/json

{
  "asset_ids": [1, 2, 3],
  "depreciation_date": "2025-03-31"
}
```

### 获取折旧记录
```
GET /api/assets/{asset_id}/depreciation/records?page=1&per_page=20
```

### 获取折旧预测表
```
GET /api/assets/{asset_id}/depreciation/schedule?years=5
```

### 获取折旧报表
```
GET /api/depreciation/report?start_date=2025-01-01&end_date=2025-03-31
```

### 获取折旧统计
```
GET /api/depreciation/statistics
```

## 常见问题

### Q: 如何选择合适的折旧方法?

A: 根据资产类型选择:
- **直线法**: 办公家具、建筑物、机械设备等价值稳定的资产
- **双倍余额递减法**: 电脑、手机、软件等技术更新快的资产

### Q: 残值如何确定?

A: 残值通常是资产原值的5%-10%,也可以参考:
- 历史报废记录
- 市场二手价格
- 税法规定的最低比例

### Q: 折旧后账面价值为负数怎么办?

A: 系统会自动控制账面价值不低于残值,不会出现负数。

### Q: 可以修改历史折旧记录吗?

A: 不可以。折旧记录一旦生成就不能修改,但可以重新配置折旧参数并重新计算。

### Q: 资产出售后如何处理折旧?

A:
1. 确认累计折旧总额
2. 确认账面价值
3. 记录出售信息
4. 财务入账处理

### Q: 折旧方法可以更改吗?

A: 可以。修改资产的折旧方法后,新折旧将按新方法计算,历史记录保持不变。
