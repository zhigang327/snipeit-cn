## 五、使用指南

### 1. 部署步骤
```bash
# 1. 运行数据库迁移
docker-compose exec backend php artisan migrate

# 2. 重启后端服务
docker-compose restart backend

# 3. 访问系统
http://localhost:5173
```

### 2. 功能使用
1. **创建维修记录**:
   - 进入"维修记录"菜单
   - 点击"新增维修记录"
   - 选择资产、填写故障描述
   - 设置优先级和类型
   - 保存

2. **分配维修人员**:
   - 在列表中找到待处理的记录
   - 点击"分配"按钮
   - 选择维修人员
   - 确认分配

3. **完成维修**:
   - 找到处理中的记录
   - 点击"完成"按钮
   - 填写故障诊断和解决方案
   - 记录实际工时和费用
   - 确认完成

4. **查看统计**:
   - 页面顶部显示统计卡片
   - 总维修记录、待处理、处理中、完成率

### 3. 配置微信通知
在 `.env` 文件中添加：
```bash
WECHAT_NOTIFY_MAINTENANCE=true
```
